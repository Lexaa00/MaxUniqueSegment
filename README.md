# Longest Substring Without Repeating Characters

## Description
This TypeScript program finds the length of the **longest substring without repeating characters** in a given input string.

- Accepts input from the console.
- Iterates through all possible substrings to identify the longest one without duplicates.
- Outputs the length of the longest substring.
